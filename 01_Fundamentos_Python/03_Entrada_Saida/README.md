# 03 - Entrada e Saída de Dados

Esta pasta contém exemplos de entrada e saída de dados em Python.

## Conceitos Abordados:
- Função `print()` e formatação de saída
- Função `input()` para entrada de dados
- Formatação de strings (f-strings, format(), %)
- Conversão de tipos na entrada de dados

## Exemplos a serem criados:
- `exemplo_print.py` - Demonstração de diferentes formas de usar print()
- `exemplo_input.py` - Entrada de dados do usuário
- `formatacao_strings.py` - Diferentes formas de formatar strings
