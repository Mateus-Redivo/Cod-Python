Lista rápida de exercícios (pastas 1 a 5)

1) Declare 5 variáveis (nome, idade, altura, estudante, cidade) e imprima tipo + valor de cada uma.

2) Troque os valores de duas variáveis sem usar variável auxiliar.

3) Faça operações aritméticas com dois números e mostre soma, subtração, multiplicação, divisão, resto e potência.

4) Verifique se um número é par ou ímpar usando operador %.

5) Compare dois números e exiba: maior, menor ou iguais.

6) Leia idade e informe se é maior de idade (>= 18).

7) Receba três notas e diga se a média está aprovada (>= 7), recuperação (>= 5) ou reprovada.

8) Converta str para int e float e trate erro caso a entrada seja inválida.

9) Converta temperatura de Celsius para Fahrenheit e Kelvin.

10) Peça nome e idade do usuário e imprima uma mensagem formatada com input + print.

11) Use operadores lógicos para validar acesso: usuário correto e senha correta.

12) Crie uma condição com if/elif/else para classificar número: negativo, zero ou positivo.
